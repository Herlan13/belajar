Cara Install XAMPP di Windows

Ini adalah kelanjutan artikel sebelumnya membuat website sendiri, yang dimana untuk membangun sebuah website berbasis php mysql dibutuhkan webserver dan untuk membuat webserver local kita akan menggunakan XAMPP, karena program ini memiliki paket yang lengkap terdiri dari Apache 2.2.21, MySQL 5.5.16, PHP 5.3.8, phpMyAdmin 3.4.5, FileZilla FTP Server 0.9.39, Tomcat 7.0.21 (with mod_proxy_ajp as connector) untuk lebih jelasnya anda bisa kunjungi websitenya (http://www.apachefriends.org/en/xampp-windows.html). Dengan adanya paket dari XAMPP ini akan memudahkan kita membuat website dengan platform PHP. Dan kelebihan dari XAMPP ada PHP switch didalamnya dari php versi 5 ke versi 4 atau dari versi 4 ke 5. Bagi anda yg belum memiliki XAMPP silahkan download XAMPP 1.7.4 for windows disini.

Install XAMPP di windows sangat mudah sekali anda lakukan, tinggal klik next dan oke XAMPP sudah terinstal di komputer anda. Berikut ini adalah beberapa langkah meng-install XAMPP di windows.

Install XAMPP step 1

Jalankan programnya kemudian klik next, jangan klik cancel hehehe

Tentukan drive tempat anda akan menyimpan hasil extrac dari XAMP. Jika sudah klik next

Install XAMPP step 2

Jika muncul tampilan seperti gambar di bawah, centang saja semua

Install XAMPP step 3

Kemudian akan muncul proses extrack file XAMPP, tunggu sampai proses selesai, kemudian tinggal next/ok, jika semua proses berjalan dengan lancar maka akan muncul XAMPP Control Panel Application, klik start pada bagian kanan Apache dan Mysql sehinggal status menjadi Running.

FINISH

Proses installasi XAMPP berakhir. gampang kan? silahkan cek dengan mengetikkan http://localhost pada browser anda. Jika muncul seperti gambar di bawah berarti proses install xampp berhasil dan berjalan dengan baik,
