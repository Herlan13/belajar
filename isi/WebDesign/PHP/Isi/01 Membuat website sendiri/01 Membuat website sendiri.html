Membuat website sendiri

Untuk memulai membuat website ada beberapa hal yang diperlukan, misalnya seperti web text editor, web server, software pengolah gambar.  Yang dimana nantinya semua hal tersebut akan sangat berhubungan ketika kita membuat website, berikut ini saya jelaskan sedikit tentang hal-hal yang diperlukan untuk membuat website.

 

1. Web text editor

Karena kali ini kita akan belajar membuat website dengan Adobe Dreamweaver jadi kita perlu menginstal Adobe Dreamweaver di komputer kita. Jika anda belum memiliki softwarenya anda bisa download di website resminya dan pastinya tidak gratis, atau minta dengan teman, kalau mau anda juga bisa minta langsung dengan saya dengan cara langsung datang ke rumah saya ehehhehe  pasti saya kasi.

2. Web server

Untuk belajar membuat website secara offline tentukan kita membutuhkan server local (localhost), software pendukung untuk membuat server local misalnya Appserv dan XAMPP, tapi saya menganjurkan anda untuk memakai XAMPP karena software ini paketnya sangat lengkap untuk mendukung membuat website di localhost.

3. Software pengolah gambar.

Untuk mengolah semua gambar yang ada pada website yang akan kita buat nantinya anda bisa menggunakan software Adobe Photoshop, saya yakin anda sudah memiliki software yang satu ini, jika belum anda bisa mendownload di website resminya, saran saya sih minta kepada teman lebih gratis tis tis, orang Indonesia gitu loooh, sukanya yg gratis gratis. Sebenarnya banyak software pengolah gambar, tapi saya sangat menyarankan anda menggunakan Adobe Photoshop karena software ini sudah terkenal, hahahaha bukan itu alasan utamanya, alasannya karena software ini memiliki tool yang lengkap dan fitunya juga sangat lengkap untuk mendukung membuat website. Seperti nantinya kita akan membuat desain layoutnya terlebih dahulu, membuat background, membuat logo website, membuat desain header, dan lain - lain.

Karena pembahasan membuat website ini cukup panjang maka saya memutuskan untuk memecahnya menjadi beberapa halaman, intinya yang penting anda sabar dan tekun pastinya anda bisa membuat website anda sendiri.
