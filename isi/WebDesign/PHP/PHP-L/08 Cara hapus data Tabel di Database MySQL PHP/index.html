Setelah kita membuat form untuk menambah data, saatnya kita untuk belajar cara hapus data tabel di database MySQL dengan skrip PHP. Pada tutorial ini kita akan belajar membuat formulir untuk operasi hapus data dari tabel MySQL.

Data dapat dihapus dari tabel MySQL dengan mengeksekusi SQL DELETE melalui fungsi PHP mysql_query. Untuk lebih jelas tentang perintah SQL untuk delete data, kunjungi tutorial DELETE Syntax.

Di bawah ini adalah contoh skrip sederhana untuk menghapus data dari tabel karyawan. Untuk menghapus data di setiap tabel, diperlukan satu kolom data yang akan digunakan sebagai klausa bersyarat. Contoh di bawah ini menggunakan data pada kolom id_karyawan yang telah disetting menjadi primary key untuk mencocokkan baris data dalam tabel karyawan.
