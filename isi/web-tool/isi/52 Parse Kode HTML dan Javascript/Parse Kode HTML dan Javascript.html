Cara menggunakan Parser HTML Code Tool :
  1. Paste kode (HTML, PHP, Java Script, dll) yang ingin di parse pada form yang ada.
  2. Klik tombol "Parse HTML Code" dan kode yang ada akan secara otomatis muncul dengan kode yang benar.
  3. Klik tombol "Block All" agar kode hasil parse ter-block semua /terpilih untuk bisa di salin (copy dan paste) di tempat Mas Bos meletakkan kode-kode tersebut.
  4. Jika anda ingin memulai parse kode baru / yang lain, Klik "Reset" agar form / kolom kembali kosong.
Semoga bermanfaat (",)
