Completa colección de herramientas CSS para diseñadores web

  • Editor Gradientes

    Usa varios colores y opacidades para obtener magníficos gradientes.

    Las imágenes generadas pueden usarse como imágenes de fondo en fondos de pantalla, botones y en muchos otros elementos.

  • Bordes Redondeados

    Super sencillo de usar y ahorra mucho tiempo. Cambia todos los bordes seleccionados al mismo tiempo.

    Crea sutiles fondos con algo de suciedad y de ruido, cambiando el color del fondo y de la suciedad, y viendo el resultado en tiempo real.

  • Editor Sombras

    Cambia el radio de la sombra, el color, tamaño… Todo lo que necesitas para crear sombras en un solo lugar.
