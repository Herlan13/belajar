Multidirectional Gradient

Use transparent layered background to create multigradient because gradients in CSS3 make use of color stops. web browsers don't recognize multiple backgrounds. Make one gradient per layer.

	background: -ms-linear-gradient(270deg, rgba(111,151,210,0.55) 0%, rgba(135,183,255,1) 50%, rgba(111,151,210,1) 51%, rgba(111,151,210,0.6) 100%);
